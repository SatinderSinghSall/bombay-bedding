# Bombay Bedding — Flagship Digital Showroom

A premium, highly-optimized production storefront web application engineered for **Bombay Bedding**, Durgapur's premier luxury home furnishings landmark established in 1962. This system delivers an immersive digital curation of bespoke fabric assemblies, boutique textiles, luxury bedding linens, and high-fidelity local discovery features.

---

## 🏗️ System Architecture & Framework Stack

The platform is designed around cutting-edge frontend patterns maximizing rendering efficiency, accessibility (a11y), and discovery mechanics:

### Core Framework Infrastructure

- **Next.js 16.2 (App Router):** Leverages optimal Server Component strategies alongside React 19 concurrent architectural lifecycles for high initial page speed metrics.
- **React 19 & React DOM 19:** Utilizes native streaming capabilities and performance optimizations.
- **TypeScript 5.7:** Enforces strict compile-time safety and self-documenting static type guarantees across structural UI properties.

### User Interface & Layout Engineering

- **Tailwind CSS v4.0:** Utilizes utility-first semantic composition with modern declarative styling layers.
- **Framer Motion 12.0:** High-fidelity, multi-axis entry orchestrations, physics-based micro-interactions, and spatial animation flows.
- **Lucide React:** Monochromatic, highly-scalable dynamic vector iconography matching modern luxury typography design systems.
- **Shadcn/UI Custom Primitives:** Clean design system component primitives configured on top of `class-variance-authority` and `@base-ui/react`.

### Performance & Telemetry Engine

- **Vercel Web Analytics Engine:** Production-level user interaction observation framework tracking web vitals dynamically without inflating initial chunk weights.
